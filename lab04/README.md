# Лабораторная работа 4
## Реализуемый паттерн
**Plugin (Дополнительный модуль)** 

## Идея
Программа моделирует работу кассового терминала кофейни. В зависимости от выбранного плагина (бонуса) итоговая сумма заказа рассчитывается по‑разному:

- **StandardBonus** – стандартная накопительная скидка (5%).
- **BirthdayBonus** – скидка в честь дня рождения (50%).
- **StudentDiscount** – скидка для студентов (10%).
- **HappyHourDiscount** – скидка в «счастливый час» (20% в определённое время).
- **FifthCoffee** – 5-ый кофе в подарок (100%).

| Компонент | Тип | Назначение |
|-----------|-----|-------------|
| `PluginManager` | class | Класс для взаимодействия пользователя с плагинами |
| `Bonus` | интерфейс | Интерфейс для определения общего поведения бонусов |
| `StandardBonus` | class | Стандартный бонус |
| `BirthdayBonus` | class | Бонус в день рождения |
| `StudentDiscount` | class | Студенческая скидка |
| `HappyHourDiscount` | class | Счастливый час |
| `FifthCoffee` | class | 5-ый подарочный кофе |
