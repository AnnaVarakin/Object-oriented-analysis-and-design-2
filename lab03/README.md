## Суть

Программа представляет собой симуляцию муравьиной колонии, где каждый муравей выполняет свою специализированную роль, а вся координация происходит через матку, которая выступает в роли посредника.

### Основные участники:
- **5 типов муравьёв** с уникальным поведением:
  - **Няньки (NannyAnt)** — ухаживают за личинками
  - **Строители (BuilderAnt)** — расширяют и ремонтируют гнездо
  - **Разведчики (ScoutAnt)** — ищут новые источники еды и открывают территории
  - **Сборщики (ForagerAnt)** — собирают и приносят еду
  - **Солдаты (SoldierAnt)** — патрулируют территорию и защищают колонию
- **Матка**: управление муравейником

## Реализация паттерна

| Компонент | Реализация | Описание |
|-----------|-------------------|----------|
| **Mediator** (интерфейс) | `IMediator` | Определяет метод `send` для обмена сообщениями между коллегами |
| **ConcreteMediator** | `QueenAnt` (Матка) | Реализует логику координации: получает сообщения от муравьёв и раздает задания, обновляет ресурсы, принимает решения о создании новых муравьёв |
| **Colleague** (абстрактный класс) | `Ant` | Определяет общий вид всех муравьев |
| **ConcreteColleagues** | `NannyAnt`, `BuilderAnt`, `ScoutAnt`, `ForagerAnt`, `SoldierAnt` | Конкретные типы муравьёв с уникальным поведением |
